สร้างแบบฟอร์มสำหรับเว็บไซต์ HTML
ตั้งชื่อให้กับ fields ทุกอัน
ชี้ action ไปยัง formspree.io
แล้วยืนยันอีเมลของคุณ

ขั้นตอนโดยละเอียด

  1. เชื่อมต่อแบบฟอร์ม
    <form action="https://formspree.io/ใส่อีเมลของคุณตรงนี้" method="POST" /> สังเกตข้อความในเครื่องหมายคำพูด แทนที่ข้อความ *ใส่อีเมลของคุณตรงนี้* ด้วยอีเมลของคุณ เช่น
    <form action="https://formspree.io/nip@gmail.com" method="POST" />
  2. เพิ่มคุณลักษณะ name attributes ในทุกๆ ส่วนประกอบ เช่น แท็ก input, select, และ textarea ดังตัวอย่างต่อไปนี้ <input type="text" name="name">
  3. ส่งและยืนยัน
    ส่งแบบฟอร์มครั้งแรก แล้วนั่นจะเป็นการยืนยันการขอใช้งานแบบฟอร์มนี้

ตัวอย่างแบบฟอร์มติดต่อทางอีเมล (Email Contact Forms)

<form method="POST" action="https://formspree.io/ใส่อีเมลของคุณตรงนี้"> <input type="text" name="name" placeholder="ชื่อ - นามสกุล"/> <input type="email" name="email" placeholder="อีเมลของคุณ"> <textarea name="message" placeholder="ความคิดเห็นของคุณ"></textarea> <button type="submit">ส่งความคิดเห็น</button> </form>

Google Fonts

โดย นิป เอมรัฐ
๑๑ ก.พ. ๒๕๖๒

Google Fonts เสนอฟอนต์คุณภาพสูงสำหรับใช้กับเว็บไซต์ของเรา

ข้อดี

  1. ใช้ง่าย
    เพียงเพิ่งลิงค์ของ stylesheet ลงไปในเว็บเพจของเรา แล้วเรียกใช้ฟอนต์ CSS style
  2. ฟรี
    ใช้ได้ฟรีทั้งในเว็บที่ไม่หารายได้ (non-commercial) หรือในเว็บทางการค้า (commercial)

Google Fonts. (2019, Feb 11). Retrieved from https://developers.google.com/fonts/

การเลือกฟอนต์

  1. https://fonts.google.com/
  2. Search > Languages > Thai
  3. ทดลองดูฟอนต์ในรูปแบบต่างๆ เช่น Sentence, Paragraph, Alphabet เป็นต้น
  4. ทดลองดู ปรับต่อไปตามต้องการ เช่น Regular, Light, Thin เป็นต้น รวมถึงขนาดฟอนต์ต่างๆ
  5. ทดลองพิมพ์ข้อความลงในกล่องข้อความ เพื่อดูผลลัพธ์ว่าจะตรงตามต้องการหรือไม่
  6. คลิกเครื่องหมายบวกที่มุมบนขวา เพื่อเลือก สมมติว่า เราเลือก Pattaya และ Krub

การนำฟอนต์ไปใช้กับเว็บเพจ

  1. เปิดลิ้นชักฟอนต์ที่เลือกไว้
  2. ฝังฟอนต์ (Embed Font) โดยนำรหัสที่ได้ไปวางไว้ภายในแท็ก head <link href="https://fonts.googleapis.com/css?family=Krub|Pattaya" rel="stylesheet">
  3. ระบุชื่อฟอนต์ในไฟล์ CSS font-family: 'Pattaya', sans-serif; font-family: 'Krub', sans-serif;

Google Fonts. (2019, Feb 11). Retrieved from https://fonts.google.com/

API คืออะไร?

API เป็นคำย่อของ “Application Programming Interface.”

API เหมือนกับเมนูในร้านอาหาร เมนูจะบอกเราว่ามีอาหารอะไรบ้างที่เราสั่งได้ และบอกรายละเอียดเกี่ยวกับอาหารแต่ละจายด้วย เมื่อเราระบุถึงอาหารที่เราต้องการแล้ว พ่อครัวก็จะปรุงอาหารนั้นให้เรา เราไม่รู้ว่าทางร้านเตรียใอาหารนั้นอย่างไร เราไม่จำเป็นต้องรู้

เช่นเดียวกัน API จะแสดงรายการทำงานที่นักพัฒนาสามารถเรียกใช้ได้ พร้อมทั้งคำอธิบายถึงสิ่งที่มันกระทำ นักพัฒนาไม่จำเป็นจ้องรู้ถึงวิธีการที่มันทำงาน เช่น ระบบปฏิบัติการสร้างและแสดงไดอะล็อกบ็อกซ์ “Save As” เราเพีงแต่รู้ว่ามีมันอยู่ และสามารถเรียกใช้ได้ในแอปนั้น

How to Geek. (2019, Feb 11). What Is an API? Retrieved from https://www.howtogeek.com/343877/what-is-an-api/

สมัครสมาชิก Github

  1. ไปที่ github.com
  2. Signup
    เลือก Username*
    ใส่อีเมล์
    รหัสผ่านอย่างน้อย 8 ตัว ประกอบด้วยตัวอักษรภาษาอังกฤษตัวเล็กและตัวเลข
    *Username ที่เลือกมีความสำคัญมาก เพราะมันจะปรากฏอยู่ในที่อยู่ (URL) ของเว็บไซต์ของคุณด้วย
  3. Verify account
    ทำแบบทดสอบ เพื่อทดสอบว่าเราเป็นคนจริงๆ
  4. เลือกแผน Free > Continue
  5. ขั้นสุดท้ายเป็นแบบสอบถาม สามารถข้าม (skip this step) ไปได้
  6. เปิดอีเมล เพื่อยืนยันที่อยู่อีเมล คลิกปุ่ม Verify email address
  7. ที่หน้าเริ่มต้น คลิก Start a project
  8. ตั้งชื่อในช่อง Repository name ดังนี้ "ชื่อผู้ใช้ของคุณ.github.io"
  9. คลิกปุ่ม Create repository
  10. คลิกที่ README
  11. เปลี่ยนชื่อไฟล์จาก README.md เป็น index.html
  12. ลบข้อมูลเดิมออก แล้วแทนที่ด้วยข้อมูลจากไฟล์ index.html ของคุณ ด้วยคำสั่ง Copy > Paste
  13. คลิกปุ่ม Commit new file เพื่อบันทึกการทำงาน
  14. ดูผล โดยพิมพ์ที่อยู่เว็บของคุณในเว็บเบราว์เซอร์ "ชื่อผู้ใช้ของคุณ.github.io" หากไม่เห็นหน้าเว็บ ให้รอสักครู่ แล้วทำซ้ำ จะเห็นหน้าเว็บภายในเวลาไม่เกินสิบนาที
  15. คลิกปุ่ม Upload files เพื่ออัพโหลดไฟล์อื่นๆ ที่เหลือขึ้นไปให้ครบ
  16. เลือก choose your files
    เลือกไฟล์ที่ต้องการ
  17. เมื่อไฟล์ถูกอัพโหลดขึ้นไปหมดแล้ว คลิกปุ่ม Commit changes
  18. โหลดหน้าเว็บซ้ำเพื่อดูผลลัพธ์ความเปลี่ยนแปลงที่ได้
    โดยกดแป้น F5 (reload) หรือ Shift+F5 (hard reload) เพื่อล้างหน่วยความจำของเว็บเบราว์เซอร์ (browser cache)
